Перед тем как использовать удаленные репозитории в git необходимо установить прокси при помощи следующих команд:

git config --global http.proxy http://proxyuser:proxypwd@192.168.251.1:8080
git config --global https.proxy https://proxyuser:proxypwd@192.168.251.1:8080

где необходимо заменить учетные данные для доступа к прокси серверу на свои
